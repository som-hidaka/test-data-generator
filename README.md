# テストデータ生成ツール マニュアル

## 概要

テスト用のファイルをブラウザ上で生成・ダウンロードできるツールです。
ファイルのアップロード機能などのテストに必要なデータを、環境構築不要で手軽に用意できます。

---

## 機能一覧

### 1. 通常ファイル生成

指定した形式・サイズのファイルを1件ずつ生成します。

**対応形式**

| カテゴリ | 形式 |
|--------|------|
| 画像 | JPG / PNG / GIF / WebP / BMP / SVG |
| 動画 | MP4 / AVI / MOV / MKV |
| ドキュメント | PDF / TXT / CSV / JSON / XML / HTML / DOCX / XLSX / ZIP |

**サイズ指定**
- 0バイト（空ファイル）〜10MBのプリセットから選択
- 「カスタム指定」で任意のサイズ（B / KB / MB）を指定可能

**画像サイズ指定**（画像形式選択時のみ表示）
- 自動：100×100pxの小さい画像を生成
- カスタム：幅・高さをピクセル単位で指定
- プリセット：よく使うサイズを一覧から選択

| プリセット | 用途例 |
|-----------|------|
| 1×1 | 最小サイズのテスト |
| 1920×1080 (FHD) | 標準的な画像サイズ |
| 3840×2160 (4K) | 大きい画像のテスト |
| 10000×10000 | 超大解像度のテスト |
| 1×10000 / 10000×1 | 極端なアスペクト比のテスト |

**画像オプション**
- **透過**：アルファチャンネルを持つ透過画像を生成
- **EXIFデータ付き**：位置情報などのメタデータを含む画像を生成
- **アニメーションGIF**：2フレームのアニメーションGIFを生成（GIF選択時のみ有効）

**その他オプション**
- **破損ファイル**：中身がランダムなバイト列の壊れたファイルを生成
- **長いファイル名**：255文字のファイル名を生成
- **特殊文字ファイル名**：`@` `#` `$` `%` などを含むファイル名を生成
- **日本語ファイル名**：日本語のファイル名を生成

---

### 2. 拡張子偽装ファイル生成

中身と拡張子が異なるファイルを生成します。
「JPG画像のデータなのに拡張子が .png」といったファイルで、サーバー側のファイル検証ロジックのテストに使用します。

**操作手順**
1. 「実際の中身」で本来のファイル形式を選択
2. 「偽装する拡張子」で表示上の拡張子を選択
3. 「リストに追加」をクリック

**例）** 実際の中身：`EXE` → 偽装する拡張子：`jpg`
→ 実行ファイルのヘッダーを持ちながら、拡張子が `.jpg` のファイルが生成される

---

### 3. 一括生成

複数のファイルをまとめて生成します。

#### サイズ範囲指定

開始〜終了サイズを刻み幅で分割したファイルを一括生成します。
容量制限のギリギリ前後を一気に用意したい場合などに便利です。

**操作手順**
1. ファイル形式を選択
2. 開始サイズ・終了サイズ・刻み幅をそれぞれ入力
3. 「プレビュー」で生成件数と一覧を確認
4. 「全てリストに追加」をクリック

**例）** JPG、1KB〜10KB、刻み1KB → 1KB / 2KB / 3KB ... 10KB の10件を生成

#### パターンリスト

形式・サイズの組み合わせを自由に複数行で定義して一括生成します。
特定のテストシナリオに必要なファイルセットをまとめて用意したい場合に便利です。

**操作手順**
1. 行ごとにファイル形式・サイズ・備考を入力
2. 「＋ 行を追加」で行を増やせる
3. 「全てリストに追加」をクリック

---

### 4. 生成リストとダウンロード

追加したファイルは「生成リスト」に一覧表示されます。

- 不要なファイルは「削除」ボタンで個別に削除可能
- 「まとめてダウンロード（ZIP）」で全ファイルをZIPに圧縮してダウンロード
- 「リストをクリア」で全件削除

---

## よくある使い方

### アップロード機能のテスト

| テスト内容 | 設定例 |
|-----------|------|
| 対応形式の確認 | 各形式のファイルを1件ずつ生成 |
| 非対応形式の拒否確認 | 対象外の形式を生成してアップロード |
| 容量制限の確認 | 一括生成（範囲指定）で制限前後のファイルを生成 |
| 拡張子チェックの確認 | 拡張子偽装ファイルを生成してアップロード |
| 空ファイルの処理確認 | サイズ0バイトのファイルを生成 |
| 破損ファイルの処理確認 | 破損オプションをONにして生成 |
| 特殊ファイル名の処理確認 | 特殊文字・日本語・長いファイル名オプションを使用 |

---

## 注意事項

- 生成されるファイルはテスト用途のダミーデータです。実際に開けるファイルではありません（画像は除く）
- 大きいサイズや高解像度の画像を大量に生成するとブラウザの動作が重くなる場合があります
- ダウンロードはすべてZIP形式です。解凍してから使用してください
